Positioning Example

I am positioned relatively.
I am positioned absolutely.
body { font-family: Arial, sans-serif; background-color: #f0f0f0; padding: 20px; } .parent { position: relative; /* Set the parent to relative positioning */ width: 300px; height: 200px; border: 2px solid #007BFF; padding: 10px; background-color: #fff; margin: 20px auto; } .relative-positioned { position: relative; /* Relative positioning */ top: 20px; /* Move down by 20px from its original position */ left: 30px; /* Move right by 30px from its original position */ background-color: #28a745; padding: 10px; color: white; border-radius: 5px; } .absolute-positioned { position: absolute; /* Absolute positioning */ top: 10px; /* Move down from the parent */ left: 10px; /* Move right from the parent */ background-color: #dc3545; padding: 10px; color: white; border-radius: 5px; }
