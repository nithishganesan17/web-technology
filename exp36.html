This is an example of using a background image with external CSS.

body { margin: 0; font-family: Arial, sans-serif; } .background-container { height: 100vh; /* Full viewport height */ background-image: url('https://example.com/path/to/your/image.jpg'); /* URL of your background image */ background-size: cover; /* Cover the entire container */ background-position: center; /* Center the background image */ color: white; /* Text color */ display: flex; /* Use Flexbox for centering content */ flex-direction: column; /* Stack elements vertically */ justify-content: center; /* Center vertically */ align-items: center; /* Center horizontally */ text-align: center; /* Center text */ }
